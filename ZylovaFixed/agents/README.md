# Zylova Agent Templates

Ready-to-use AI agent system prompts for every industry.
Customize each template with your business details and drop it
into your Zylova website in under 10 minutes.

## Quick Start

1. Open the .md file for your industry
2. Copy the SYSTEM PROMPT section
3. Open your website's js/agent.js
4. Replace SYSTEM_PROMPT content with your customized version
5. Fill in all [PLACEHOLDER] values with your real info
6. Test it on your website before going live

## Tips for Great Agents

- Be specific about pricing ranges (not exact — markets vary)
- Include your actual hours, not generic placeholders
- Add your real booking URL so the agent can direct customers
- Name your owner/manager so the agent can personalize escalations
- Test with 10+ real scenarios before launching
- Review the agent's conversations weekly and update the prompt
